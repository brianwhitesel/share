Rocket League Clip Organizer Live Tool

A keyboard-driven tagger for bulk clip batches. Load a folder, watch each clip play automatically, press a key to assign a category — the tool builds the filename in real time and exports a full rename map when you're done.

  • Action categories: Goal, Save, Demo, Aerial, Dribble, Assist, Freestyle + more
  • Context detail buttons auto-populate based on the category selected
  • Keyboard shortcuts — 1–9 for categories, Enter to advance, Space to play/pause
  • Live filename preview on every clip before it's committed
  • Export modal with full rename map — ready to batch-rename in one step
  • Naming convention: 260430_aerial_flip_reset_goal.mp4
